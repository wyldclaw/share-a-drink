# Share a Drink

Show achievement progress on tooltips for [[Share a Drink]](https://www.wowhead.com/achievement=61081/share-a-drink) when mousing over players in the Arcantina.

Be sure to purchase enough [Toasting Brew](https://www.wowhead.com/item=251039/toasting-brew) from [Bartender Bob](https://www.wowhead.com/npc=250495/bartender-bob).

It can be hard to remember which races you've shared a Toasting Brew with. Instead, avoid tracking achievements or keeping the panel open with this helpful addon.
